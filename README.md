## Install

```shell
npm install pepebecker/papago-cli
```

Make sure these environment varibles are configured on your system

```shell
export NAVER_CLIENT_ID='<naver-client-id>'
export NAVER_CLIENT_SECRET='<naver-client-secret>'
```
